# Caesar Cipher

Simple and easy to use Caesar Cipher.
## Functionalities

- Encode
- Decode


## Demonstration

```csharp
// Hello to Ifmmp
Caesar.Encode("Hello", 1)

// Ifmmp to Hello
Caesar.Decode("Ifmmp", 1)

```
